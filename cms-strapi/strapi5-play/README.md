# code playgrounds for strapi v5

# quickstart

```shell
# start strapi admin webapp
npm run develop
```

- open a browser and go to http://localhost:1377/admin

- integrated plugins
  - custom-fields: editor-quill, table-nosontable, select-multi
  - version-trail: version history for strapi built-in collections
  - watoffice
